        O Projeto IFSC exibe lembretes antes do prazo das tarefas cadastradas.

          Sobre

          Projeto IFSC foi criado para ajudar estudantes universitários e professores a organizar tarefas, prazos e atividades acadêmicas.

          • Cadastro de usuário com e-mail e senha.
          • Login seguro e fácil de usar.
          • Gerenciamento de tarefas com título, descrição e prazo.
          • Edição e exclusão de tarefas em poucos cliques.
          • Lembretes automáticos antes do vencimento.

          Esta versão funciona inteiramente no navegador usando HTML, CSS e JavaScript com armazenamento local.

          Projeto IFSC - Organização acadêmica simples e acessível.
